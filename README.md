# 空爆シミュレーター (Air Strike Simulator)

Pythonのコンソール上で動作する、シンプルな2次元グリッド式の空爆シミュレーションプログラムです。
指定した座標を中心に、円形の爆撃範囲（ユークリッド距離）を計算し、範囲内にあるターゲットを破壊する仕組みをシミュレートします。

## 🛠 機能特徴

* **2次元グリッドマップ:** `10x10`（サイズ変更可能）の仮想マップ上にターゲットを配置。
* **正確な範囲判定:** 三平方の定理（ユークリッド距離）を用いた、半径指定による円形爆撃。
* **視覚的なコンソール表示:** 絵文字（Emoji）を使って、爆撃エリアや着弾判定を視覚的に表現。

## 📋 動作環境

* Python 3.x 以降
* 外部ライブラリのインストールは不要です（標準ライブラリ `math` のみ使用）。
